### 정렬

- 선택정렬
  - 앞에서부터 순서대로 확인하며 가장 작은 값을 앞으로 swap
  - 시간복잡도 `O(N^2)`

```python
# 선택정렬 
# 가장 작은 값을 앞에서부터 확인하며 swap하여 정렬한다

min = 0
index = 0
temp = 0
ary =[1,10,5,8,7,6,4,3,2,9]
for i in range(10):
    min = 9999
    for j in range(i, 10):
        if min > ary[j]:
            min = ary[j]
            index = j
            
    temp = ary[i]
    ary[i] = ary[index]
    ary[index] = temp
print(ary)
```
